\hypertarget{systems-of-linear-equations}{%
\chapter{Systems of Linear
Equations}\label{systems-of-linear-equations}}

\begin{tcolorbox}[enhanced jigsaw, opacitybacktitle=0.6, rightrule=.15mm, bottomrule=.15mm, colbacktitle=quarto-callout-note-color!10!white, leftrule=.75mm, toptitle=1mm, titlerule=0mm, coltitle=black, colback=white, opacityback=0, bottomtitle=1mm, title=\textcolor{quarto-callout-note-color}{\faInfo}\hspace{0.5em}{Definition of a linear equation}, breakable, arc=.35mm, left=2mm, toprule=.15mm, colframe=quarto-callout-note-color-frame]

A linear equation in the variables \(x_1, x_2, \ldots, x_n\) is an
equation that can be written in the form

\[a_1x_1 + a_2x_2 + \cdots + a_nx_n = b\]

where \(b\) and the coefficients \(a_1, a_2, \ldots, a_n\) are real or
complex numbers. The subscript \(n\) may be any positive integer.

\end{tcolorbox}

\begin{tcolorbox}[enhanced jigsaw, opacitybacktitle=0.6, rightrule=.15mm, bottomrule=.15mm, colbacktitle=quarto-callout-note-color!10!white, leftrule=.75mm, toptitle=1mm, titlerule=0mm, coltitle=black, colback=white, opacityback=0, bottomtitle=1mm, title=\textcolor{quarto-callout-note-color}{\faInfo}\hspace{0.5em}{Definition of a system of linear equations}, breakable, arc=.35mm, left=2mm, toprule=.15mm, colframe=quarto-callout-note-color-frame]

A system of linear equations (or a linear system) is a collection of one
or more linear equations involving the same variables- say
\(x_1, x_2, \ldots, x_n\). An example is

\[
\begin{align*}
&2x_1 - x_2 &+ 1.5x_3 &= 8 \\
&x_1 &- 4x_3 &= -7
\end{align*}
\]

\end{tcolorbox}
